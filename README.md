Информационная система «Аппетитная Доставка» 🍕
Описание проекта
Данный проект представляет собой десктопное приложение для автоматизации работы службы доставки еды. Система позволяет пользователям просматривать меню и оформлять заказы, а администраторам — управлять ассортиментом, отслеживать логи и изменять статусы заказов.

Проект выполнен в рамках учебной работы по объектному проектированию и разработке информационных систем.

Ключевой функционал
Для клиентов:

Авторизация и регистрация в системе.

Интерактивное меню с фильтрацией и поиском блюд.

Удобная корзина с расчетом итоговой стоимости.

Оформление заказа с указанием адреса и комментария.

Для администраторов:

Управление каталогом (добавление, редактирование, удаление блюд).

Мониторинг всех заказов и смена их статусов (new -> cooking -> delivering -> delivered).

Просмотр системных логов событий для аудита безопасности.

Управление профилями пользователей.

Технологический стек
Язык программирования: Python 3.x

Библиотека интерфейса: CustomTkinter (современный UI в стиле Dark/Light mode)

База данных: SQLite3 (реляционная модель из 5 таблиц)

Архитектура: Объектно-ориентированное программирование (ООП)

Структура базы данных
Система базируется на нормализованной базе данных, включающей таблицы:

users — учетные данные и роли.

dishes — каталог доступных блюд.

orders — заголовки заказов.

order_items — детализация состава заказов.

logs — журнал системных событий.
